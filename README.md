# Insights e Visualizações - Top 100 Spotify Brasil

## Sobre o Projeto

Este projeto foi desenvolvido como trabalho da disciplina de Visualização de Dados e apresenta análises visuais sobre as músicas, artistas e gêneros mais populares do Spotify no Brasil, nos anos de 2021-2023.

## Objetivo

Explorar e visualizar dados do Top 100 do Spotify Brasil entre os anos de 2021 e 2023, identificando padrões, tendências e insights sobre o comportamento musical dos brasileiros.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

## Período Analisado

2021 - 2023
